How to Install and Use the Android App

Step 1: Download the App

Tap the button below to start downloading the app file (APK):

Download App

If asked, choose "Download anyway" or "Keep file" if your phone gives a warning that the file might be harmful. Please don’t worry — this is a standard security warning that Android shows for apps not downloaded from the Play Store. It does not mean the app is unsafe or contains a virus.

Step 2: Allow App Installation

When downloaded, go to My Files app on your phone, and click the apk file. Your phone may not allow you to open the file or may block the installation since it’s not from the Play Store. Follow these steps:

  1. Go to Settings on your phone.
  2. Search for Install unknown apps.
  3. Select and Turn on for the My Files app. To allow from this source
  4. Go back to My Files on your phone.
  5. Find the recently downloaded apk file and click on it to proceed with the installation.

Step 3: Install the App

  1. When prompted and the installer opens, tap Install.
  2. Wait for it to finish, then tap Open.

Step 4: Give Location Access

When the app opens for the first time, it will ask for GPS location permission.

  • Please choose Allow all the time.
  • If asked, please turn on GPS/location services in your phone settings.
  • If you are not asked to turn on GPS/Location please do so anyways.

Step 5: Start Your Prayer Walk

  1. Open the app and tap on Login or Register if you've previously not created an account.
  2. Onced logged in, tap Get Start to begin your walk.
  3. You’ll see your location on the map and stats like time and distance.
  4. You can pause or stop at any time. When you stop, your walk is saved.

Need Help?

If something isn’t working:

  • Make sure GPS is on and permissions are allowed.
  • Try reinstalling the app.
  • If the download won’t start, try using a different browser like Chrome.
